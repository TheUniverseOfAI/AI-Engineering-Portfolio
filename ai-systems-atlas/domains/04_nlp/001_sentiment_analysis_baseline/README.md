        ## Why it exists

        ## Folder structure
        - `src/` → code
        - `notebooks/` → exploration
        - `data/` → raw and processed data
        - `models/` → trained artifacts
        - `outputs/` → figures, metrics, logs
        - `configs/` → config files
        - `tests/` → tests
        - `scripts/` → helper run scripts

        ## Setup
        ```bash
        uv sync
        ```

        ## Run
        ```bash
        uv run python -m src.app
        ```
